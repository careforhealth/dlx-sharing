## 🌍 我们是谁

由**患者、家属、康复者**共建的互助社区。我们不是医生,但经历过同样挣扎，愿意分享经验、照亮彼此的路。

## 🎯 我们提供

- **📚 实用信息**：科学减药方法（微珠减量法）、副作用应对、品牌对比、国内外研究
- **💬 同伴支持**：分享故事、获得理解、困难时刻的鼓励
- **🛠️ 工具资源**：症状追踪表、减药计划、就医指南
- **🤝 长期陪伴**：减药是马拉松，我们陪你走完全程

## 🌟 价值观

- **相互尊重**：不评判、只倾听
- **科学为本**：分享经验但不替代医疗建议
- **隐私保护**：可匿名分享，信息保密
- **赋能自主**：帮你成为自己的专家
- **希望导向**：康复是可能的

## 📊 你需要了解的真相

我们基于**真实用户反馈**，整理了关键数据：

### 戒断反应的真实情况
根据社区问卷调查（92位朋友参与）：
- **50%以上** 经历中度到极度的戒断痛苦
- **27.2%** 经历极度痛苦（每4人就有1人）
- **仅18.5%** 感受"没感觉"
- **医学说明书声称"少见"** ❌ 与真实数据严重不符

### 为什么会这样
- 度洛西丁半衰期只有 **12小时** —— 是最难停的抗抑郁药之一
- 需要采用 **双曲线递减法**（每次≤5%当前剂量）
- 可能需要 **12-24个月** 的时间才能安全停药
- 快速减半？❌ **80%以上会失败并重新出现严重症状**

### 成功的关键
✅ 科学的减药方法（不是"减半"）
✅ 足够的时间（尊重身体的适应周期）
✅ 社区支持（你不是一个人）
✅ 专业指导（在医生监督下进行）
